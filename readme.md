# Usage

## React

Install the package via [npm](https://www.npmjs.com/package/@ikono/react)

```
npm install @ikono/react
```

Then to use the icon import and insert into jsx.

```jsx
import { Home } from "@ikono/react";

const Component = () => (
  <div>
    <Home />
  </div>
);
```
